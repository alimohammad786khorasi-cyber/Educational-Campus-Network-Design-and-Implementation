# Secure Campus Network Design

## Overview

Designed and implemented a secure two-floor campus network in Cisco Packet Tracer for an educational institute. The project uses VLANs, inter-VLAN routing, DHCP, DNS, and structured IP addressing to provide secure, scalable, and efficient communication between departments.

## Features

* VLAN-based network segmentation
* Inter-VLAN routing
* DHCP for automatic IP allocation
* DNS for hostname resolution
* Structured IPv4 addressing
* Secure and scalable network design

## Technologies Used

* Cisco Packet Tracer
* Cisco Routers & Switches
* VLANs
* DHCP
* DNS
* IPv4 Addressing

## Learning Outcomes

* Campus network design
* VLAN configuration
* Inter-VLAN routing
* DHCP and DNS configuration
* IP addressing and subnetting
* Basic network troubleshooting
